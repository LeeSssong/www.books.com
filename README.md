* 项目描述
  * 使用ThinkPHP5实现前后端分离的图书管理系统
  * 主要功能
    * 登陆注册
    * 普通用户权限
      * 对于图书
        * 借阅、归还、查找
      * 对于个人信息
        * 修改密码、注销账号、挂失账号
    * 管理员权限
      * 对于图书
        * 增加、删除、修改、查找图书
      * 对于用户
        * 查看已有用户数目及其所借阅图书
        * 修改用户挂失状态
* 前端技术栈
  * Bootstrap
  * html、css、JavaScript、Ajax、json
* 后台技术栈
  * PHP
* 数据库
  * MySQL
